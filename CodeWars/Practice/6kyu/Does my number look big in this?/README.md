# Does my number look big in this?

## Условие задачи
Создать функцию, которая будет определять является ли число нарциссическим (число Армстронга).
Нарциссическое число — это натуральное число, которое равно сумме своих цифр, возведенных в степень, равную длине этого числа.

## Решение
```python
def narcissistic( value ):
    l = len(str(value))
    sum = 0
    for i in str(value):
        sum += int(i) ** l
    if sum == value:
        return True
    else:
        return False
```

## Объяснение
Сначала мы записываем длину числа в переменную, для удобства. А также задаём переменную для суммы чисел возведённых в степень. С помощью цикла for проходим по каждому элементу числа и прибавляем к sum цифру, возведённую в степень, равную длине. После этого производим проверку. Если сумма совпала с исходным числом, то это число является нарциссическим. Иначе, возвращаем "False".

## Нюансы
Вместо блока проверки if/else, можно было просто написать "return sum == value". Эта строка производит сравнение и выводит булевое значение в зависимости от результата.
