# Array.diff

## Условие задачи
Написать функцию, которая вычисляет разницу между двумя списками. Эта функция должна удалять все элементы из первого списка (а), которые присутствуют во втором перечне (b). Результат должен быть записан в список (a).

## Решение
def array_diff(a, b):
    for i in b:
        while i in a:
            a.remove(i)
    return a

## Объяснение
Цикл for проходит по всем элементам списка (b), для каждого элемента выполняется цикл while, который будет выполняться пока не будет выполнено условие. Условием же является: "пока i находится в a", выполни удаление элемента (i) из списка. Если (i) нет в списке (a), то цикл завершится и for перейдёт к следующему элементу. Так будет происходить, пока все элементы не будут удалены. В конце мы возвращаем переменную (a).

## Нюансы
Удаление элементов во время прохода по тому же списку может привести к смещению индексов и пропуску значений. Поэтому в этом решении используется внешний проход по (b) и последовательное удаление всех совпадений из (a).
